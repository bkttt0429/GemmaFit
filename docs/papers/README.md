# Papers And Research Notes

This folder stores literature review notes and research extraction work used to
support the project framing.

## Files

| File | Purpose |
| --- | --- |
| `literature_review.md` | Main literature and claim-support review. |
| `literature_review_zh.md` | Chinese working version of the literature review. |
| `senior_care_dual_task_review_zh.md` | Senior care and dual-task research notes. |
| `research_extraction_backlog.md` | Open extraction queue and unresolved research items. |

## Claim Boundary

Use these notes to support non-clinical, pose-based movement-quality feedback
only. Do not turn literature notes into product claims for diagnosis,
fall-risk prediction, sarcopenia detection, rehabilitation prescription,
clinical improvement, force/GRF estimation, or EMG-style muscle activation.
