# Assets Index

This folder contains submission and demo media assets. Keep source materials,
generated images, and final gallery outputs separated so the competition
package can be assembled quickly.

## Root Assets

| File | Purpose |
| --- | --- |
| `gemmafit_cover_1920x1080.png` | General 16:9 cover image. |
| `gemmafit_cover_deadlift_1920x1080.png` | Alternative 16:9 cover image. |
| `gemmafit_social_1200x630.png` | Social preview / card image. |
| `gemmafit_phone_1080x1920.png` | Phone-format promotional image. |
| `gemmafit_app_icon.svg` | App icon source. |

## Subfolders

| Folder | Purpose |
| --- | --- |
| `video/` | Generated video previews, contact sheets, caption files. |
| `video_sources/` | Scripts and source images used to generate evidence-chart videos. |

## Rules

- Do not store raw personal video by default.
- Prefer generated preview images, contact sheets, captions, and final
  competition-safe media.
- Keep scripts in `video_sources/` and generated outputs in `video/`.
- If a media file is only a benchmark artifact, store it under
  `docs/benchmark/<run>/` instead of this folder.
