# Demo Scripts Index

This folder stores narration, movement-command scripts, and demo video copy.

## Files

| File | Purpose |
| --- | --- |
| `gemmafit_evidence_intro_voiceover_en.md` | English voiceover for the evidence-first demo story. |

## Rules

- Keep scripts competition-safe: movement-quality language only, no diagnosis
  or clinical prediction.
- Put language in the filename when possible, for example `_zh` or `_en`.
- Keep final spoken copy short enough for the 3-minute demo video.
